# EVO Museum 

##언리얼엔진을 이용한 실감형 온라인 가상 전시 플랫폼

웹페이지 접속 
-> 언리얼엔진으로 만든 가상전시공간 접속 
-> 픽셀스트리밍 통해 실시간 생성한 렌더링 프레임 이용 
-> 가상 전시공간을 자유롭게 이동하고 작품 감상
